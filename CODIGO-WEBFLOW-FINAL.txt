INSTRUÇÕES:
===========================================

1. Acesse: Webflow → Site Settings → Custom Code
2. Carregue Locomotive Scroll, GSAP, ScrollTrigger e Finsweet (se usar listas) antes
3. Cole os scripts weldpoly acima no Footer Code
4. Clique em "Save"
5. Publique o site
